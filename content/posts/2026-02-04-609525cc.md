- AIチップ開発のスタートアップ「Positron」が、シリーズBで2億3000万ドル（約345億円）という巨額の資金調達を実施しました。

- 単なる処理能力の向上だけでなく、独自のアーキテクチャによる電力効率とコスト削減を武器に、AIインフラの民主化を目指しています。

## 何が発表されたのか

みなさん、こんにちは。AI専門ブロガーのねぎです。日々進化するAI業界ですが、今日は「ハードウェア」の分野で非常に大きな動きがありました。

今回、TechCrunchによって独占的に報じられたのは、AI向け半導体を開発するスタートアップ「Positron」が、シリーズBラウンドで2億3000万ドルもの資金調達を完了したというニュースです。日本円に換算すると約345億円。スタートアップの資金調達としては破格の規模ですよね。

この調達を主導したのは、カタールの政府系ファンドであるカタール投資庁（QIA）です。ここが非常に重要なポイントで、現在カタールは国家戦略としてAIインフラの構築に力を入れています。NvidiaのGPU（H100やB200など）が世界中で極端な品不足に陥り、さらに価格も高騰し続けている中で、特定の企業（Nvidia）に依存しない独自の計算基盤を手に入れたいという国家レベルの思惑が、今回の巨額出資の背景にはあるわけです。

Positronという会社は、これまでステルス（秘密裏）に近い形で活動してきましたが、今回の調達を機に一気に表舞台へと躍り出ました。彼らの目標は明確で、「Nvidiaの代替となる、より効率的で安価なAIチップを提供すること」です。

現在の生成AIブームは、凄まじい計算リソースを消費します。OpenAIのSoraや、次世代のGPT-5（仮）のような大規模モデルを動かすには、数千、数万台のGPUが必要です。しかし、Nvidia一強の状態では、供給が不安定でコストも膨大です。Positronはこの「Nvidia一強」という構造的なボトルネックを解消するための急先鋒として期待されています。

投資家たちがこれほどまでにPositronに期待を寄せるのは、彼らが提示しているビジョンが「単なる追随者」ではなく、「AI時代に最適化された全く新しい計算構造」を提案しているからに他なりません。元SIerのエンジニアとして、多くのサーバーを見てきた私から見ても、今回のニュースは今後のデータセンターのあり方を根本から変えてしまう可能性を秘めていると感じます。

## 技術的なポイント

さて、ここからは少し深掘りして、Positronのチップが技術的にどのような特徴を持っているのかを解説していきますね。

Positronが目指しているのは、現在の主流である「汎用的なGPU（Graphics Processing Unit）」ではなく、AI処理、特に大規模言語モデル（LLM）の推論や学習に特化した「LPU（Language Processing Unit）」や「ASIC（特定用途向け集積回路）」に近いアプローチです。

1. メモリ・ボトルネックの解消
従来のコンピューティングには「フォン・ノイマン・ボトルネック」という課題があります。これは、計算を行うプロセッサと、データを保存するメモリの間でデータをやり取りする際に、転送速度が追いつかず遅延が発生してしまう問題です。Positronのアーキテクチャは、計算回路のすぐ近くにメモリを配置する、あるいは計算回路そのものにメモリ機能を持たせるような「イン・メモリ・コンピューティング」的なアプローチを採用していると推測されます。これにより、データの移動距離を最小限に抑え、消費電力を劇的に削減しながら、処理速度を爆発的に向上させています。

2. 消費電力あたりの計算性能（ワット・パフォーマンス）
今のデータセンターにとって最大の悩みは「電力」です。Nvidiaのチップは非常に強力ですが、その分消費電力も凄まじいものがあります。Positronのチップは、AI計算に必要な演算（行列演算など）に特化し、不要な機能を削ぎ落とすことで、圧倒的なワット・パフォーマンスを実現しようとしています。これは、持続可能なAI開発において極めて重要な要素です。

3. ソフトウェアとの統合
Nvidiaが強い最大の理由は、ハードウェアそのものよりも「CUDA」というソフトウェア・プラットフォームにあります。多くの開発者がCUDAに慣れ親しんでいるため、他社のチップに乗り換えるのは非常に手間がかかります。Positronはこの点において、既存のPyTorchやTensorFlowといったフレームワークからスムーズに移行できるような、高度なコンパイラ技術を開発しているといいます。

4. スケーラビリティ
Positronの設計は、単体のチップとして優れているだけでなく、数千個、数万個のチップを相互に接続して「一つの巨大なコンピュータ」として動かす際の通信効率が最適化されています。モデルが大規模化し続ける現状において、この「チップ間の通信速度」こそが、AIの学習速度を左右する鍵となります。

このように、Positronは単に「性能が良い」だけではなく、コスト、電力、そしてソフトウェアの使い勝手という、実運用上の課題を解決しようとしているのが技術的な強みだと言えます。

## 競合との比較

今回のPositronの発表を、皆さんもよく知るChatGPT（OpenAI）やClaude（Anthropic）などの「AIモデル・サービス」と比較して整理してみましょう。ここでの注意点は、Positronは「ハードウェア」であり、ChatGPTなどはその上で動く「ソフトウェア（サービス）」であるという点です。

| 項目 | 今回の発表 (Positron) | ChatGPT (OpenAI) | Claude (Anthropic) |
|------|-----------|---------|--------|
| カテゴリ | 半導体（ハードウェア） | AIモデル・チャットアプリ | AIモデル・チャットアプリ |
| 主な役割 | AIを動かすための「エンジン」 | ユーザーとの対話、テキスト生成 | ユーザーとの対話、コード生成 |
| 依存先 | 半導体製造工場 (TSMC等) | Nvidia GPU / Microsoft Azure | Nvidia GPU / AWS |
| 課題 | CUDAからの移行、量産体制 | 計算コストの高騰、供給不足 | 独自の計算資源の確保、コスト |
| 今回の意義 | Nvidia依存からの脱却手段 | 将来的に安価に動かせる可能性 | 将来的に安価に動かせる可能性 |

Positronは、ChatGPTやClaudeを「より安く、より速く動かすための土台」を作っています。

現在、OpenAIもAnthropicも、その運営コストの大部分をNvidiaへの支払いや、Nvidia製チップを並べたクラウドサーバーの利用料に費やしています。Positronのような競合が現れることで、ハードウェアの価格競争が起きれば、最終的には私たちが利用するChatGPTの有料プランが安くなったり、無料版の制限が緩和されたりといったメリットに繋がるわけです。

また、ChatGPTやClaudeは「汎用的な知能」を目指していますが、Positronは「その知能を物理的に支える筋肉」を鍛えているようなイメージです。どんなに優れた頭脳（ソフトウェア）があっても、それを動かす筋肉（ハードウェア）が脆弱だったり高価すぎたりすれば、社会への普及は進みません。その意味で、PositronはAI業界の「インフラ・レボリューション」を担っている存在だと言えます。

## 業界への影響

このPositronの台頭と巨額増資は、AI業界全体にどのような影響を与えるのでしょうか。短期的・長期的な視点で分析してみます。

短期的には、「Nvidia一強」への牽制（けんせい）として機能します。
現在、AI開発企業にとってNvidiaは「絶対的な王者」であり、その価格設定や供給量に逆らうことはできません。しかし、Positronのような実力派スタートアップに巨額の資金が流入することで、Microsoft、Google、Amazonといったメガクラウド企業以外の「第三の選択肢」が生まれます。これにより、特定の企業にリソースを握られるリスク（ベンダーロックイン）を避けたい企業にとって、非常に魅力的な選択肢となるはずです。

また、今回の出資元がカタールであるという点も無視できません。これは「ソブリンAI（国家によるAIの自律性）」の動きを加速させます。自国でエネルギー（電力）を持ち、自国でハードウェア（Positron等）を確保し、自国のデータでAIを育てる。この流れが中東だけでなく、世界各国で加速するでしょう。

長期的には、AIの「推論コスト」の劇的な低下が期待されます。
今、私たちは1回ChatGPTに質問するたびに、数円から数十円の計算コストがかかっていると言われています。これがPositronのようなAI特化型チップによって10分の1、100分の1になればどうなるでしょうか。あらゆる電化製品やIoTデバイスに高度なAIが標準搭載され、24時間365日、コストを気にせずAIが私たちの生活をサポートする世界が現実味を帯びてきます。

一方で、懸念点もあります。それは「エコシステムの壁」です。先ほども触れましたが、NvidiaのCUDAはあまりにも普及しており、エンジニアにとって他のチップを使うことは心理的・技術的なハードルが高いのです。Positronが成功するかどうかは、単にチップの性能が良いというだけでなく、いかにエンジニアに「これならNvidiaじゃなくてもいい、いや、こっちの方が使いやすい」と思わせる開発環境を構築できるかにかかっています。

このPositronの挑戦は、AIという火を灯し続けるための「新しい薪」を探すようなものです。もしこれが成功すれば、AIの進化スピードはさらに一段階ギアが上がることは間違いありません。

## 私の見解

ここからは、私「ねぎ」の個人的な感想を少しお話しさせてください。

正直なところ、今回のニュースを聞いた時、私は「ようやく来たか！」というワクワク感と同時に、「これはNvidiaにとっても相当なプレッシャーになるな」と確信しました。

エンジニア時代、私はサーバーの調達やセットアップも担当していましたが、ハードウェアの世界は非常に保守的です。性能が2倍になったからといって、信頼性や互換性が不明な新しいチップに飛びつくことは稀です。それでも今回、Positronが2億3000万ドルという金額を集められたのは、それだけ現在の「Nvidia依存」に対する業界の危機感が強いことの裏返しだと言えるでしょう。

個人的には、Positronのような企業にはぜひ頑張ってほしいと思っています。というのも、今のAI業界は少し「パワープレイ」に寄りすぎている気がするからです。巨大な資本を持つ企業が、大量のGPUを買い占めて、力技でモデルを大きくしていく。このやり方では、一部の強者しかAIの恩恵をフルに受けられません。

Positronが目指す「高効率・低コスト」なハードウェアが普及すれば、スタートアップや個人の開発者でも、もっと自由に、もっと大規模なAIを試行錯誤できるようになります。私がブログで紹介したいのは、そういう「技術が民主化される瞬間」なんです。

もちろん、Nvidiaが黙って見ているはずもありません。彼らも次々と新しいチップを発表していますし、ソフトウェアの壁も依然として高いです。Positronにとっての戦いは、資金調達を終えたここからが本番でしょう。

みなさんは、AIの進化においてハードウェアの重要性をどう感じていますか？ 普段使っているAIの裏側で、こうした「チップの戦争」が起きていると思うと、少し見え方が変わってきませんか。Positronが今後、実際のベンチマークスコアや、初期の導入事例をどう発表してくるのか。私も専門ブロガーとして、引き続き目を光らせてウォッチしていきたいと思います。

皆さんもぜひ、これからのAIハードウェアの動向に注目してみてくださいね。
