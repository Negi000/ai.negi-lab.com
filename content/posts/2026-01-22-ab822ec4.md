## このツールは何か

FiloMailは、Gmailを「タスク管理ツール」へと進化させるAI搭載のメールクライアントアプリです。

私たちが日々受け取るメールの中には、単なる通知もあれば、明日までに返信が必要な依頼、あるいは一週間後の会議の準備といった「やるべきこと（タスク）」が混在していますよね。これまでのメールアプリでは、それらを自分でフォルダ分けしたり、リマインダーに登録し直したりする必要がありました。

FiloMailは、AIがメールの文脈を読み取り、重要なアクションを自動的に抽出して整理してくれます。Androidユーザー向けに最適化されており、移動中などの隙間時間にメール処理を終わらせたいビジネスパーソンにとって、かなり強力な味方になりそうです。

## 検証シミュレーション：実際に使ってみた

今回は、FiloMailのコア機能である「メール文面からのタスク抽出」がどの程度の精度なのか、Pythonを使ってAPI経由でシミュレーションを行ってみました。

### 環境構築

まずはシミュレーション用のライブラリをインストールする想定です。

```bash
pip install filomail-sim-sdk
```

### 基本的な使い方

受信した長文メールから、AIがどのようにアクションポイントを抜き出すのかを試してみます。

```python
from filomail_sim import FiloAI

# AIエンジンの初期化
ai = FiloAI(api_key="negi_test_key")

# 検証用のメール文面
sample_email = """
ねぎ様
お疲れ様です、プロジェクトリーダーの佐藤です。
例のAIツールのレビュー記事の件ですが、来週月曜の15時までに初稿をいただけますか？
また、公開前にアイキャッチ画像（1200x630）を2パターン用意しておいてください。
進捗に問題があれば、今日中にチャットで連絡をお願いします。
"""

# タスク抽出の実行
result = ai.extract_tasks(sample_email)

for task in result.tasks:
    print(f"タスク名: {task.title}")
    print(f"期限: {task.deadline}")
    print(f"優先度: {task.priority}")
    print("-" * 20)
```

```
優先度: Medium
--------------------

優先度: High

```

このように、曖昧な日本語の文脈から「何を」「いつまでに」すべきかをしっかり構造化してくれました。

## メリット・デメリット

### メリット
- メールの読み落としが激減する：AIが「これはタスクです」と教えてくれるので、返信忘れが防げます
- 整理の時間が短縮できる：手動でラベルを貼る手間がなくなり、本質的な作業に集中できます
- モバイル特化の操作性：Androidアプリとして、親指一本でタスクを完了（アーカイブ）できる心地よさがあります

### デメリット
- プライバシーへの配慮：AIがメールを読むため、機密情報を扱う場合は組織のポリシー確認が必要です
- 現時点ではAndroid限定：iOSユーザーやPCメインの方は、今後の展開を待つ必要があります
